ヘルプ

  1. 一覧には今日を含む未来日の申請が昇順で表示されます。
  2. 検索フォームの期間、申請者、ステータスを条件に、任意の届書を検索することができます。
  3. 一覧は期日で昇順／降順ソートをすることができます。
  1. 届書の申請は、申請画面から行います。
  2. 申請すると、申請者本人と決裁者へメールが送信されます。
  3. 申請した内容を変更することはできません。
  4. 申請した内容に誤りがある場合は、取下げ、または却下したうえで、新しく申請し直します。
  5. 一度取り下げた申請は元に戻すことは出来ません。
  1. 申請者の上長（決裁者）は申請者の代理で届書を申請することができます。
  2. 代理申請を行う場合、申請画面の代理申請欄に、代理で申請したい申請者を選択してください。
  3. メールは通常の申請通り、申請者の決裁者の双方に送られます。
  1. 申請者が届書を申請すると、決裁者にメールが届きます。
  2. 決裁者はメールに記載された URL にアクセスするか、一覧画面から検索し詳細画面を開きます。
  3. 詳細画面では、決裁者にのみ承認／却下ボタンが表示されます。
  4. 承認／却下すると、決裁者本人と申請者へメールが送信されます。
  1. 設定画面にて、パスワードの変更を行うことができます。
  2. パスワードの有効期限は 3 ヶ月です。パスワードは定期的に変更してください。
  3. パスワードの期限が切れるとパスワード変更以外の操作が制限されます（ログインは可能です）。
  1. 【承認済みのみを表示する】 チェックを入れると届書一覧を表示したとき承認済みのもののみ表示されます。 この設定は検索には反映されません。検索は任意のステータスで行うことが出来ます。
  1. 管理権限を持つユーザーは管理画面を閲覧することができます。
  2. 管理画面では、ユーザーの登録／編集／削除／有効化／無効化を行うことができます。
  3. いずれかの申請が存在するユーザーは削除できません（誤登録に対する救済機能）。
  4. 無効化されたユーザー（退職した社員を想定）はログイン不可となります。
  5. 決裁者を変更しても申請済みの届書の決裁者は変更されません。
